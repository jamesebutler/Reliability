• Verification section added to Reliability Incident (RI) Enter/Update Incident page
• Verification Default will be No
• Responsible person dropdown will refresh based on Facility selection for the incident
• No Special security for Verification task on MTT
• To add a verification task for an incident:
• Select Yes in Verification section
• Enter Responsible person and Weeks After (weeks after analysis complete date)
• Click on Save Reliability Incident button
• MTT task will be created
•  Due date will be populated as RI Analysis Complete date + Weeks After (due date will be blank until the Analysis Complete date has been set)
• Click  Update Verification Task button to navigate to MTT Task Detail page for updating  comments, Closed Date, etc.
•To remove a verification task for an incident:
• Select No in Verification section
• Click on Save Reliability Incident button
